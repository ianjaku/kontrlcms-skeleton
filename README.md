# KontrlCMS Skeleton

This is a good base to start off with when you're creating a website.

## Install

```
git clone https://github.com/ianjaku/kontrlcms-skeleton.git
```

```
composer install
```


## Running the php server

You can use the php dev server for local development

```
cd public

php -S localhost:8000
```

## Running webpack

This skeleton comes with a fully set up webpack server with an scss parser.
It's not required and you can just delete it if you don't want it.

```
cd assets

npm run watch
```
